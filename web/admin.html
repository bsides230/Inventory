Rebuild Inventory from Excel

Drop your updated English Master.xlsx and Spanish Master.xlsx files into the item master/ folder on the server, then click the button below to rebuild the inventory data.

Excel Format

  • Each sheet tab = one inventory category
  • Column A = item names (one per row, no header)
  • English and Spanish masters must have matching sheet names
  • Sheet tab names become category names in the app

Location PINs

Each location uses a 4-digit PIN to identify themselves when placing orders. The PIN and location name are stamped on every order log entry.
