Upload Master Inventory File

Upload a single Master.xlsx file to update the inventory. The existing file is automatically backed up as a .bak file before replacement.

Master.xlsx Format

  • Each sheet tab = one inventory category
  • Column A = English item names (no header row)
  • Column B = Spanish item names (same row order)
  • Embed an icon in the tab name using Category Name (🍕) or Category Name "🍕"
  • The icon is extracted automatically and shown in the app
Example tab names: Produce (🥦) Meats (🥩) Beverages (🥤)
